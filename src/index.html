WebApp-Menu

Easy and accessible context menus and toolbars.

WebApp-Menu is a set of stand alone web-components for use in web applications. It provides elements for toolbars and pop-up menus. These elements are accessible for mouse and keyboard interaction. They can be styled to match the look of your application.

You can download WebApp-Menu from NPM or from GitHub.
